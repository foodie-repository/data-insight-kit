# Report Rubric

도메인 보고서가 반드시 포함해야 할 내용을 정의한다.

## Summary Report

- 한 줄 결론
- 핵심 발견 3개 이내
- 주요 근거
- 해석 주의
- 다음 행동

## Deep Report

- 의사결정 질문
- 방법론과 데이터 한계
- KPI 정의
- 세그먼트/분포/관계/추세 분석
- 반대 해석과 리스크
- 실행 시나리오
- 추가 분석 설계
- lineage 부록

## 독자별 톤

| 독자 | 강조점 | 피해야 할 것 |
|---|---|---|
| executive | 판단과 리스크 | 계산식 과다 노출 |
| analyst | 방법론과 재현성 | 근거 없는 결론 |
| operator | 실행 기준과 체크리스트 | 추상적 제안 |
